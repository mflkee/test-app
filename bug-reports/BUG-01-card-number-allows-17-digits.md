# BUG-01. Поле номера карты принимает 17 цифр

## Краткое описание

После ввода 17 цифр сервис считает номер карты валидным и открывает следующий шаг с вводом суммы перевода, хотя по требованиям номер карты должен состоять ровно из 16 цифр.

## Серьёзность

`Major`

## Предусловия

- Сервис открыт по URL `http://localhost:8000/?balance=30000&reserved=20001`

## Шаги воспроизведения

1. Нажать на карточку `Рубли`
2. Ввести в поле номера карты `12345678901234567`

## Ожидаемый результат

- Поле ограничивает ввод 16 цифрами
- Следующий шаг с вводом суммы не отображается

## Фактический результат

- В поле остаётся 17 цифр
- Отображается блок `Сумма перевода`

## Повторяемость

`100%`
